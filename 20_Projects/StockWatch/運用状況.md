# StockWatch 検証運用を開始

項番004の検証運用を開始。アプリ名 StockWatch を構築済み。

- 場所: `/home/slosea/lifework/stockwatch/`
- 技術スタック: FastAPI + SQLite + APScheduler + Jinja2
- データソース:
  - edinetdb.jp(機関投資家・アクティビスト動向、無料100req/日)
  - J-Quants API v2(株価、無料5req/分・12週遅延)
- 検出シグナル:
  - 新規アクティビスト参入
  - 保有比率の+1%以上の増加
  - 同一業種への報告集中
- 毎日05:00(JST)に自動収集(EDINETの24時更新後)
- アクセス: LAN内全端末から `http://192.168.10.142:8000`
- 未対応: systemd常駐化のため `sudo systemctl enable --now stockwatch` を別途実行する必要あり
- 初回実行結果: 200件取得、22シグナル検出(電気機器セクター集中、AXIUM CAPITAL/ヨウテイHDなどの新規参入)
